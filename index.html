🚀 Progressive Web App

Demonstração prática dos conceitos de PWA

📱 Manifest.json

Arquivo responsável por transformar o site em um aplicativo instalável.

  • Nome do aplicativo
  • Ícones
  • Cor do tema
  • Tela inicial
  • Modo Standalone

⚡ Service Worker

Script executado em segundo plano.

  • Controla o cache
  • Funciona offline
  • Intercepta requisições
  • Melhora a performance

💾 Cache Storage

Armazena arquivos localmente.

  • HTML
  • CSS
  • JavaScript
  • Imagens

📊 Estratégias de Cache

  • Cache First → Mais rápido
  • Network First → Mais atualizado
  • Cache Only → Apenas cache

🌐 Offline First

Após visitar a aplicação uma vez, os arquivos ficam salvos localmente.

Mesmo sem internet o usuário poderá acessar a aplicação.

📲 Instalação

Uma PWA pode ser instalada no celular ou computador sem passar pela loja de aplicativos.
